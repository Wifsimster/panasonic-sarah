Introduction

Ce plugin SARAH s'appuie sur le travail effectué par THLabs sur le module node-panasonic-viera.
Ce module Node.js permet de contrôler les TV Panasonic Viera via requête HTTP avec un message XML.

Configuration du plugin SARAH

Il vous suffit de préciser l'adresse IP (Hostname) de votre TV Panasonic Viera, ou le nom par défaut COM-MID1.
Pour information, si vous avez une Freebox V5 il n'y a pas de résolution de nom local sur votre réseau, donc passer directement à l'adresse IP.

Commandes possibles

L'ensemble des commandes fournis par le module Node.js sont implémentées dans le plugin SARAH.

Commandes :
  • Eteindre la TV (pour l'allumer, je passe pas une prise Chacon DIO)
  • Gestion du son (on/off, volume)
  • Gestion des chaines TV (up/down, TF1, France 2, ... jusqu'à la chaine 14 de votre TV)
  • Affichage du menu et sous-menu
  • Gestion des boutons rouge, vert, jaune et blue de la télécommande Panasonic
  • Gestion du lecteur vidéo (play, pausen stop, arrière, avant et enregistrement)
  • Affichage des sous-titres
  • Passage en mode 3D
  • Affichage du guide, menu texte et informations
  • Navigation dans les menu (retour, annule, entrer, gauche, droite, monte, en haut, descend et en bas)

Pour connaitre les commandes exactes, mercide vous repporter au fichier panasonic.xml.
